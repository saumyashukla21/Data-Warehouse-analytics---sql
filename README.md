# Data Warehouse Analytics - SQL

A SQL project analyzing sales, customer, and product data using a data warehouse structure.

## Project Structure

- **Schema/** — Database and table creation scripts
- **Data/** — Sample CSV data (customers, products, sales)
- **Queries/** — Analysis queries including:
  - Cumulative analysis
  - Data segmentation
  - Part-to-whole analysis
  - Performance analysis
  - Customer reporting
  - Trends over time

## How to Use

1. Run the scripts in `Schema/` to set up the database and tables.
2. Load the CSV files from `Data/` into the corresponding tables.
3. Run any query from `Queries/` to explore the analysis.

## Tools Used

- SQL PostgreSQL
